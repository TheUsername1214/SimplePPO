# Simple Parallel PPO
A Simple parallel PPO Framework(Continuous & Discrete), inculding all the necessary features. 

### ————————————requirement————————————————###########

Python 3.9 and higher


Torch (no spcific version requirement)

Numpy (no spcific version requirement)

Matplotlib (no spcific version requirement)



The PPO dictionary contains continuous version of PPO. The jumping robot and inverted pendulumn dictionaries provides two discrete PPO examples. If you are serching for continusos PPO examples, please go to my "SF_TRON_xxx" repository
